Подготовка рабочей среды
1.Установить xampp из архива проекта на диск C (файл xampp-windows-x64-8.1.1-2-VS16-installer.exe)
2.Установить python из архи проекта на диск C, по окончанию установки просто нажать кнопку Close (файл python-3.10.1-amd64.exe)

Работа с конфигурацией
1.В конфигурации xampp заменить порт c 80 на 8000 (путь к файлу C:\xampp\apache\conf\httpd.conf) (строчки #Listen 12.34.56.78:8000 и Listen 8000)
2.Запустить xampp
3.В программе xampp запустить модули Apache и MySQL
4.Извлеч папку проэкта в папку установленной программы xapmm ( путь к папке xampp C:\xampp\htdocs)
5.Зайти в папку проекта laravel-fullcalender (путь к папке C:\xampp\htdocs\laravel-fullcalender)
6.Ипортировать базу данных из корня проекта (файл с расширением .sql) в phpmyadmin  (http://localhost:8000/phpmyadmin/)
   Необходимо перейти по указаному адресу в phpmyadmin. Далее нажать импорт и выбрать файл из корня проекта c расширением .sql
7.Запустить в корне проэкта исполняемый файл ip_addres.py (не закрывать появившуюсь админ панель websockets) а также в появившемся окне появится ваш IP.
  Его необходимо скопировать или записать для дальнейшего доступа к приложению. После чего можно нажать OK.
8.В браузере для доступа к приложению использовать ссылку http://IP адрес сервера:8000/laravel-fullcalender/public/
  В IP адрес сервера необходимо подставить свой локальный IP адрес который ранее был получен. 

Дополнение: если в коммандной строке не работает php необходимо прописать к нему путь.
1.Перейти в Мой компьютер->Свойства->Дополнительные параметры системы->Параметры среды->Системные переменные(создать новую переменную с именем Path и значением C:\xampp\php)
